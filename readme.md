Student Grade Manager

1. Overview
   
   This is a python-based application which helps teachers or any other member of the organization to manage student's academic records.

2. Features

   (i) CRUD Operations: We can create, read, update, and delete records.

   (ii) Persistence: Student data is saved automatically to a .json file.

   (iii) Search: You can use his to quickly find a student(s) by their ID.

4. Technologies Used

   (i) Python (3.13.9)

   (ii) JSON module (Built-in)

   (iii) OS module (Built-in)

5. How to Run

   (i) Make sure that python is installed in your PC

   (ii) Open terminal in the project folder.

   (iii) Run the command  "python main.py"
